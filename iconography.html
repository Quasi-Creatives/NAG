Ikonografi

Ikoner stödjer kommunikation — de ersätter den inte. Vår ikonstil är ren, geometrisk och ändamålsenlig.

Ikonstil

NAG använder en enlinjetjocklek outlined-ikonstil. Alla ikoner ritas på ett 24×24px-rutnät med 1,5px stroke. Hörn använder 2px-radie för ett omtänksamt, inte kallt, utseende.

EgenskapVärde
Rutnät24×24px
Stroksvikt1.5px
StilOutlined (ingen fyllning)
Hörnradie2px
CapstilRund
Rekommenderat bibliotekPhosphor Icons (Light-vikt) — eller motsvarande

Storlekar

Ikoner får bara användas i dessa storlekar. Skala aldrig till ett mellenvärde.

StorlekAnvändningsfall
16pxInlinedikoner i text, tabellåtgärder, kompakt UI
20pxNavigationsobjekt, knappdikoner
24pxStandard / default ikonstorlek
32pxFunktionslyft, kort
48pxHjälteillustration, tomma tillstånd

Användningsregler

Gör så
  • Para ikoner med synlig textetikett där utrymme finns
  • Använd en enda ikonstil i ett gränssnitt
  • Håll ikoner pixeljusterade vid små storlekar
  • Använd SVG-format för alla digitala placeringar
Gör inte så
  • Blanda outlined och filled-ikoner i samma kontext
  • Använd ikoner som dekoration utan funktionell mening
  • Lägg till skuggor eller gradienter på ikoner
  • Rotera ikoner för att skapa ny betydelse

Tillgänglighet

Varje ikon som används utan synlig etikett måste ha ett tillgängligt aria-label eller paras med visuellt dold text. Förlita dig aldrig på en ikon ensam för att förmedla kritisk information.
