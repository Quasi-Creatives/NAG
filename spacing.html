Spacing

Mellanrum är inte tomrum — det är struktur. En konsekvent spacingskala skapar rytm, hierarki och rum att andas.

Skalan

All spacing baseras på en 4px-basenhet. Använd bara dessa steg. Använd aldrig godtyckliga värden som 7px eller 13px.

--space-1 4px / 0.25rem Ikonpadding, mikrogap
--space-2 8px / 0.5rem Tight inlinespacing
--space-3 12px / 0.75rem Etikettgap, liten padding
--space-4 16px / 1rem Standard komponentpadding
--space-6 24px / 1.5rem Kortpadding, stackgap
--space-8 32px / 2rem Avsnittsspacing (tight)
--space-12 48px / 3rem Avsnittsspacing (standard)
--space-16 64px / 4rem Sidpadding, avsnittsgränser
--space-24 96px / 6rem Hjälteavsnitt, stor vertikal rytm
--space-32 128px / 8rem Hjältesektion topppadding

Spacingprinciper

Regel 1
Relaterade saker sitter närmare
Element som hör ihop använder tätare spacing. Orelaterade grupper använder bredare gap. Närhet kommunicerar samhörighet.
Regel 2
Var generös med vertikalt utrymme
Nordisk design andas. Avsnitt ska aldrig kännas trångbodda. Tveka på om du ska ta bort ett spacingsteg — lägg hellre till.
Regel 3
Konsekventa horisontella marginaler
Sidnivå horisontell padding: --space-16 (64px) på desktop, --space-6 (24px) på mobil.

Ladda ner tokens

nag-tokens.css

En färdig CSS-fil med alla färg-, typografi- och spacing-tokens. Klistra in i ditt projekt och börja använda variablerna direkt.

/* Importera i din CSS */
@import url('nag-tokens.css');

/* Sedan använd variablerna */
.hero {
  background: var(--nag-kol);
  color: var(--nag-vit);
  font-family: var(--nag-font-display);
}
