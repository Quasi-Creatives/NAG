Kontrast & tillgänglighet

Alla textkombinationer måste uppfylla WCAG 2.1 AA: 4,5:1 för normal text, 3:1 för stor text. Kol (#000000) på Ljus pistage (#F6FFD9) klarar AAA (19:1). Undvik mörk text direkt på Pistage (#D4E3A2) — kontrasterna är otillräckliga för liten text.
Gör så
  • Kol text på Ljus pistage eller Vit bakgrund
  • Vit text på Kol bakgrund
  • Pistage som accentfärg på Kol bakgrund
  • Grön-grå som bakgrund med Kol text (stor text)
Gör inte så
  • Liten Kol text direkt på Pistage (underkänt kontrast)
  • Vit text på Pistage (underkänt)
  • Grön-grå text på Pistage bakgrund
  • Pistage som textfärg — det är en bakgrundsfärg

Ladda ner tokens

nag-tokens.css

En färdig CSS-fil med alla färg-, typografi- och spacing-tokens. Klistra in i ditt projekt och börja använda variablerna direkt.

/* Importera i din CSS */
@import url('nag-tokens.css');

/* Sedan använd variablerna */
.hero {
  background: var(--nag-kol);
  color: var(--nag-vit);
  font-family: var(--nag-font-display);
}
