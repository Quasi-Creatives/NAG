Knappar

Två knapptyper — primär och sekundär. Primär används för huvudsaklig uppmaning till handling. Sekundär för alternativa eller understödda åtgärder.

Gör så
  • Primär knapp för den viktigaste åtgärden per sektion
  • Sekundär som komplement till primär
  • Håll knapptext kort: 2–4 ord max
  • Använd verb: "Boka möte", "Läs mer", "Kontakta oss"
Gör inte så
  • Använd inte fler än en primärknapp per sektion
  • Ändra inte färger, typsnitt eller padding
  • Använd inte "Klicka här" eller "Submit" som text
  • Lägg inte till ikoner utan godkännande

/* ─── Bas ─── */
.btn {
  display: inline-flex;
  font-family: 'GD Grio', sans-serif;
  font-size: 1rem;
  font-weight: 400;
  padding: .875rem 2rem;
  border-radius: 2px;
  transition: border-radius .2s ease;
  text-decoration: none;
  cursor: pointer;
}
.btn:hover { border-radius: 5px; }

/* ─── Primär ─── */
.btn-primary {
  background: #D4E3A2; /* Pistage */
  color: #000000; /* Kol */
  border: 2px solid transparent;
}

/* ─── Sekundär (mörk bakgrund) ─── */
.btn-secondary {
  background: transparent;
  color: #ffffff;
  border: 2px solid #ffffff;
}
